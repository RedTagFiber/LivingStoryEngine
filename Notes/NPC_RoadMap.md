# NPC System Roadmap

## Goal

Create NPCs that feel alive.

Every NPC should have:

- Personality
- Goals
- Needs
- Fears
- Wants
- Memories
- Relationships
- Decisions
- Thoughts

---

# NPC Base Class

Every NPC should inherit from Character.

Examples:

- Eve
- Maya
- Jake
- Mom
- Boss

# NPC Daily Update System

Each new day:

NPC evaluates:

- Mood
- Goal
- Need
- Fear
- Relationships

Example:

Day 15

Jake

Mood:
Stressed

Decision:
Call off work

- Promotion
- Overtime
- Vacation

- New Friend

- Car Trouble

- Mood

# Social Network

NPCs interact without Ryan.

Examples:

Maya calls Jake

Jake visits Mom

Boss promotes Eve

These events continue even when Ryan is not involved.

---

# Long-Term Goal

NPCs should be able to:

Remember

Feel

Change

Decide

Act

Grow

without script-driven behavior.

The world should continue moving even when the player does nothing.
